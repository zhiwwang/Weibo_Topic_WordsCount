# Weibo_Topic_Count
抓取微博话题下所有博文内容及统计词语出现频率
 
使用 Selenium 模拟用户点击操作抓取，获取对应话题下所有的博文内容。加入文本清洗，统计微博话题下对应的博文中各个词语的出现频次，可用于舆情管理，市场分析等领域。
 
PS: 由于是使用Selenium模拟真实用户操作，抓取速度较慢，建议用于少于1000条博文的中。

运行环境要求：
1. Firefox 浏览器
2. 安装驱动GeckoDriver
